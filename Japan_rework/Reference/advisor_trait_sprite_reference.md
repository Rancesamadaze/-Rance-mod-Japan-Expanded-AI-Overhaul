# HOI4 顾问特质图标编号参考

> 目的：记录 `common/country_leader/*.txt` 中 `sprite = xx` 与顾问图标外观的大致对应关系。
>
> 说明：本表优先记录本项目内已经实际核对过、已经使用过的编号，作为编写角色顾问时的速查参考。

## 当前已确认的编号

| sprite 编号 | 图标类型 | 参考角色 / 特质 | 备注 |
| --- | --- | --- | --- |
| `13` | 通用政治顾问图标 | 八意永琳、福本和夫等多数政治顾问 | 项目内最常见的默认顾问图标 |
| `14` | 理论家图标 | 山本五十六、魅魔 | 适用于 `theorist` 一类顾问 |
| `15` | 特工 / 反间谍图标 | 尾崎秀实、铃仙·优昙华院·因幡 | 适用于情报、反间谍、特工倾向顾问 |
| `10` | 海军相关顾问图标 | 帝国海军造船官、工会主义者等已存在特质 | 当前项目中常用于海军或部分特殊顾问 |

## 已在本项目中确认的用法

- 八意永琳：`sprite = 13`
- 魅魔：`sprite = 14`
- 铃仙·优昙华院·因幡：`sprite = 15`
- 尾崎秀实：`sprite = 15`

## 使用建议

- 如果只是普通政治顾问，默认可先考虑 `sprite = 13`
- 如果顾问类型是 `theorist`，优先考虑 `sprite = 14`
- 如果顾问效果明显偏向情报、反间谍、特工领域，优先考虑 `sprite = 15`
- 如果不确定，先查原版同类顾问，再决定是否沿用同一编号
