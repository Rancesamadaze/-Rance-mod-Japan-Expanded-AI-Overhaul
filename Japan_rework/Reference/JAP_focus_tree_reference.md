# 日本国策树 ID 参考

> 来源：`common/national_focus/japan.txt`
> 用途：整理日本国策树各分支的国策 ID，方便规划国策顺序和填写 `ai_national_focuses`。

## 使用说明

- 本文件只整理国策 ID 和分支顺序，不替代国策本地化对照表。
- 本文件只记录当前 NCNS / Japan rework 国策树；无 NCNS DLC 时使用的旧版兼容国策树不纳入整理。
- 中文国策名以 `Reference/focus_name.md` 为准。
- 除非小节特别说明，代码块内的 ID 顺序保持 `common/national_focus/japan.txt` 中的脚本顺序。
- 填写 `ai_national_focuses = { ... }` 时，只复制同一条路线下能够共存的代码块；许多政治路线彼此互斥。
- 新增或重命名国策时，同步更新本文件和 `Reference/focus_name.md`。
