|---|---|
| `<Doctrine>_mastery_gain_factor` | 单个学说精通度增长系数 |

| `experience_gain_<Unit>_training_factor` | 训练中单位经验获得系数 |
| `local_resources_<Resource>_factor` | 本地资源获取率 |
| `<Ideology>_drift_from_guarantees` | 境外独立保障产生的意识形态增长 |

### 使用建议

- 这部分最适合作为“**补充索引**”使用，不必强行打散并到全文各个大类里。
- 真正写脚本时，优先检查有没有：
