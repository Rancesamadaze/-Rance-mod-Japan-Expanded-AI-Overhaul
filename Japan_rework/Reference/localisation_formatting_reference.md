# HOI4 本地化格式控制符参考

> 本表根据项目内自制本地化网页工具界面截图整理。
>
> 目的：给编写本地化时快速查阅颜色码、换行符等常见格式控制写法。
>
> 说明：颜色名称以界面观感为准，属于实用参考，不追求严格色彩学命名。

## 常见格式控制符

| 写法 | 作用 | 备注 |
| --- | --- | --- |
| `\n` | 换行 | 常用于 tooltip、描述文本内手动分段 |
| `§R` / `§G` / `§Y` 等 | 开始颜色 | 将后续文本切换到对应颜色 |
| `§!` | 结束颜色 | 恢复默认文本颜色，防止后文串色 |

## 基本用法

- 开始颜色：`§R`
- 结束颜色：`§!`
- 示例：`§R红色文本§!`
- 换行示例：`第一行\n第二行`

## 使用建议

- 需要强警示，或需要表现共产主义、革命等红色政治意象时，可优先使用 `§R`
- 需要表现收益、增加、解锁时，优先使用 `§G`
- 需要普通强调时，优先使用 `§Y` 或 `§H`
- 需要在 tooltip 或描述里手动分段时，优先使用 `\n`
- 需要在一句话中混合多段颜色时，最后记得补 `§!`，避免后文串色
- 如果同一句里已经有大量高饱和颜色，尽量减少额外颜色码，避免可读性下降
