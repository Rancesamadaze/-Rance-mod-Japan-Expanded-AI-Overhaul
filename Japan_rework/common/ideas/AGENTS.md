# Ideas Directory Notes

## Scope

Applies to files under `common/ideas/`.

## Required References

- Project idea authoring notes: `common/ideas/Idea_Reference/idea_authoring_reference.md`
- HOI4 idea grammar quick reference: `common/ideas/HOI4_idea_grammar_for_AI_compact.md`

## Operating Rules

- Use existing file placement, top-level category, hidden idea, naming, and picture-reference conventions before adding new patterns.
- Do not invent unsupported fields when the idea grammar reference does not cover them.
- Treat the idea picture prefix-dropping rule as project-specific and verify local patterns before using full `GFX_idea_...` forms.
- Preserve UTF-8 without BOM and LF unless an existing file proves otherwise.
