# Units Agents Notes

## Scope

Applies to files under `common/units/`.

## Required Reference

Read `common/units/Unit_Reference/unit_authoring_reference.md` before adding or revising Japan special unit definitions, rookie/vanilla unit reference comparisons, or unit counter icon configuration.

## Operating Rules

- Use the vanilla unit directory and listed rookie reference mods as read-only references unless the user explicitly asks to edit them.
- Keep unit counter icon sizes and registration relationships consistent with the reference notes.
- Preserve existing file encoding and line endings unless the user explicitly asks for conversion.
