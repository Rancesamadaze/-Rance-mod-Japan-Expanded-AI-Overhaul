# 联合国策基本框架参照：
# - 本文件只放顶层 shared_focus / joint_focus 定义，不包在 focus_tree = { } 内。
# - 国家树通过 shared_focus = JAP_rework_co_prosperity_sphere_start 接入这条联合分支。
# - allow_branch 控制分支是否显示，通常只写在分支根节点或第一枚节点；
#   后续节点默认继承分支可见性，除非需要单独隐藏某个子分支。
# - available 控制当前国家能否点击；
#   joint_trigger 只筛选本次联合结算的参与成员，不替代 available。
# - completion_reward 是共同奖励；completion_reward_joint_originator 只给发起国；
#   completion_reward_joint_member 只给通过 joint_trigger 的非发起成员。
# - 后续节点优先使用 relative_position_id 接在已有节点上，避免在大树里散写绝对坐标。
