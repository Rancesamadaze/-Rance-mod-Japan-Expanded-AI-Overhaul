# History Directory Notes

## Scope

Applies to files under `history/`.

## Operating Rules

- This project maintains the primary 1936 start path. Do not preserve or add special handling for the 1939 start-date branch unless the user explicitly requests it.
- This project does not maintain missing-DLC fallback behavior. Do not add no-DLC compatibility logic or protect no-DLC branches unless the user explicitly requests it.
- Preserve UTF-8 without BOM and LF unless an existing file proves otherwise.
